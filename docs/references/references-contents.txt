# 기본 화면 구성

## 최상단
헤더: CAI | PRINT ver.2.2

## 좌측 중앙
### 툴바
1. redo/ undo 버튼: 사용자가 템플릿 편집 시 진행했던 사항들을 취소하거나 되돌림
2. LIBRARY: 사용자가 입력하거나 업로드 사진이 자동으로 모아짐
- 폴더: 사용자가 폴더째로 업로드 가능 / 문서 생성 시 한 번에 넣은 사진들은 새로운 폴더에 생성됨(폴더명=프로젝트 타이틀) / 템플릿 내 이미지 박스에서 수동으로 넣은 사진들은 'FROM DEVICE'라는 폴더에 들어감
- 폴더를 누르면 이미지들이 보여짐
3. SAVES: 사용자가 생성한 문서들이 임시저장되어 모아짐
4. NEW PROJECT: 플러스 버튼을 누르면 새롭게 문서를 생성할 수 있는 초기화면으로 돌아감. 진행 중이던 문서가 있을 경우 'SAVES'에 저장되며, 저장되었다는 내용이 사용자에게 전달됨

## 중앙
1. 바닥(캔버스): 그리드가 그어져 있는 무한 캔버스. 사용자가 잡고 이동 가능한 영역
2. 대지: 문서의 대지. 생성된/편집 가능한 문서가 대지로서 보여짐
- 생성된 문서가 없을 경우 '생성된 문서가 없습니다. 우측 패널에서 설정을 마치고 GENERATE 버튼을 눌러주세요.' 라는 안내문구가 보여짐

## 중앙 하단
1. 문서 미리보기: 생성된 문서들을 페이지별로 작게 미리볼 수 있는 보드. 수평으로 나열되어 있으며, 영역을 넘어갈 경우 <,> 버튼으로 빠르게 다음 끝쪽으로 넘어갈 수 있음.
- 현재 페이지/총 페이지가 보여야 하며, 현재 페이지의 경우 사용자가 텍스트 입력을 통해 전환할 수 있음

## 우측
1. 기능 전환 탭 (사이드바 상단 좌측): CAI의 10개 노드를 전환할 수 있는 탭으로, UI는 구현해 놓되 실제 노드 전환은 되지 않음(해당 프로젝트에선 PRINT만 구현) - 고정
2. 사이드바 열기/접기 버튼 (사이드바 상단 우측): 사이드바를 접고 펼칠 수 있는 버튼. - 고정
3. 사이드바: 문서 생성을 위한 설정 패드
- INSERT IMAGE: 이미지 삽입이 가능한 영역- 페이지 당 개수 제한 O. 폴더째로 올릴 수 있음 {VIDEO에선 이미지 삽입 영역 두 개(Start, End)-서로 전환 가능. 각각 영역에 사진 하나씩만 삽입 가능}
- PURPOSE: 문서 모드를 선택할 수 있는 영역 (REPORT / DRAWING&SPECIFICATION / PANEL / VIDEO)
- NUMBER OF PAGES: 페이지 수 설정 가능한 영역. (VIDEO에서 삭제)
- PROMPT: 사용자가 텍스트로 프롬프트를 전달하는 영역.
- 구분선: 스크롤을 위한 구분 선 - GENERATE 아래로 내려가면 가려짐
- CTA 2개: GENERATE(설정 후 누르면 AI가 문서/비디오 생성) / EXPORT(생성된 문서 추출: .jpg, .png, .pdf, .mp4)
