### VIDEO 모드 최종 흐름

```
page.tsx → POST /api/print (mode=VIDEO, images=[start, end])
  ↓
입력 검증 (MIME, 크기, 2장 확인)
  ↓
buildVideoInjectionPrompt() — 건축 제약 프롬프트 조합
  ↓
fal.storage.upload × 2 → startImageUrl, endImageUrl
  (실패 시: "[이미지 업로드 실패] HTTP {status} | {detail}")
  ↓
fal.subscribe(KLING_MODEL_ID, { image_url, tail_image_url, prompt })
  (실패 시: "[비디오 생성 실패] HTTP {status} | {detail}")
  ↓
result.data.video.url → PrintResult.videoUri
  ↓
VideoTemplate.tsx 렌더링
```
