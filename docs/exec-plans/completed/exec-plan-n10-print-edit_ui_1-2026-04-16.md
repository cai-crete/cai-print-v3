### 1단계: LIBRARY 모달 UI 개선
- [x] 2026-04-16 — 우측 상단에 정렬 뷰 셀렉터(그리드 뷰/리스트 뷰)를 추가하고 뷰 모드에 따른 렌더링 로직(썸네일/텍스트 리스트) 적용
- [x] 2026-04-16 — 모달 상단 타이틀인 'LIBRARY'에 `BEBAS NEUE` 폰트 적용
- [x] 2026-04-16 — 추가/삭제 버튼 디자인을 원형 속 아이콘 형태로 변경
- [x] 2026-04-16 — 삭제 기능: 삭제 버튼 선택 시 폴더/이미지에 옵션 토글 활성화 후, `DELETE`를 눌러 최종 삭제되도록 로직 수정 (폴더 삭제 모드에선 폴더 내부 진입 막음)
- [x] 2026-04-16 — 추가 기능: '새 폴더(N)' 생성 동작 추가 및 이미지 드롭 시 폴더의 대표 이미지 썸네일이 변경되도록 처리

### 2단계: SAVES 모달 UI 개선
- [x] 2026-04-16 — 모달 상단 타이틀인 'SAVES'에 `BEBAS NEUE` 폰트 적용
- [x] 2026-04-16 — 리스트 형태가 아닌 그리드(썸네일 위주) 형태로 고정
- [x] 2026-04-16 — [전체, REPORT, DRAWING&SPECIFICATION, 고정 디자인, PANEL, VIDEO] 탭 형태로 문서 필터링 뷰 구성 추가
- [x] 2026-04-16 — 추가/삭제 버튼 디자인을 원형 속 아이콘 형태로 변경
- [x] 2026-04-16 — 삭제 기능: 옵션 토글 표시 후 `DELETE` 클릭 시 일괄 삭제 처리되도록 액션 구현
- [x] 2026-04-16 — 추가 기능: 탭에 맞는 유형의 문서를 새로 생성하되 '전체'일 땐 REPORT로 불러오고, 기존 진행 중인 문서가 있을 경우 경고창 표출 기능 추가

### 3단계: SIDEBAR UI 정교화
- [x] 2026-04-16 — 각 타이틀 앞의 A, B, C, D 알파벳 라벨 제거
- [x] 2026-04-16 — INSERT IMAGE 섹션 우측에 있는 'REMOVE ALL' 버튼 크기 축소
- [x] 2026-04-16 — PANEL 선택 시 하위 메뉴(LANDSCAPE/PORTRAIT)가 바로 아래에 동일한 간격으로 나타나도록 간격 및 위치 조율
- [x] 2026-04-16 — INSERT IMAGE 내에 삽입된 이미지 요소를 수평으로 정렬하여 가로 스크롤 가능하게 스타일 설정
- [x] 2026-04-16 — VIDEO 선택 시 나타나는 설정값(16:9 | 720p | 8s) 컬러를 지정된 캡션 색상에 맞도록 일치화

### 4단계: TOOLBAR 크기 및 일관성 맞춤
- [x] 2026-04-16 — 상태에 따른 저장, 새 문서 버튼의 폭(width)을 위쪽 Pill 그룹의 버튼과 일치시키고, 전체 원형(또는 Wrapper)가 이를 감쌀 수 있도록 크기를 확장시킴

### 5단계: PREVIEW BAR 및 CANVAS 동작 최적화
- [x] 2026-04-16 — PREVIEW BAR 주변의 흰색 배경 필드를 없애고 생성된 문서가 캔버스 위에 고정된 프리뷰 페이지(줌 불가능) 형태로 존재하게 스타일링 수정
- [x] 2026-04-16 — 문서가 없을 때 표출되던 '생성된 문서가 없습니다' 텍스트 제거
- [x] 2026-04-16 — 프리뷰할 문서가 없을 때는 `<, >` 이동 버튼도 노출되지 않도록 조건부 렌더링 처리
